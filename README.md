# Vertebral Column ML Classification

Binary classification of normal vs abnormal spinal conditions 
using biomechanical measurements.

## Dataset
UCI ML Repository – Vertebral Column (310 patients, 6 features)

## Methods
- Exploratory Data Analysis (correlation, distributions)
- Logistic Regression (accuracy: 80.2%, F1: 0.843)
- Decision Tree (accuracy: 77.8%, F1: 0.841)
- 5-fold stratified cross-validation
